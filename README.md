## 📖 Overview

CodeFusion is a lightweight, fast, and beautifully designed online compiler that allows developers, students, and programming enthusiasts to write, compile, and execute code directly from their browser.

The application provides a distraction-free coding experience with a powerful Monaco Editor, real-time code execution using the Judge0 API, syntax highlighting, dark/light themes, language switching, customizable editor panels, and persistent local storage.

Unlike traditional IDEs, CodeFusion requires **no installation**, making it perfect for quick testing, coding interviews, learning programming, and competitive programming practice.

Clone the repository

```bash

```

Move into the project


OR

